Hello World!

This is the most basic example of Jcrop implementation. Since no event handlers are attached, it's not very exciting, but displays the out of box functionality. Attach it to any image on your page with the simple code below, and start cropping! 

$(function(){ $('#jcrop_target').Jcrop(); });
